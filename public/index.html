PlaylistAI

Descubra novas músicas automaticamente com base no seu gosto musical

Descrição

O PlaylistAPI é uma aplicação web que utiliza a API do Spotify para criar playlists personalizadas. O usuário informa algumas músicas de seu gosto, e o sistema, utilizando uma LLM, gera automaticamente uma playlist com faixas semelhantes em estilo, ritmo e gênero.

Funcionalidades Principais

  • Login com conta Spotify (OAuth)
  • Busca de músicas pelo nome ou artista
  • Geração automática de playlist baseada nas músicas selecionadas
  • Visualização das músicas da playlist
  • Salvar playlists geradas diretamente na conta do usuário

Tecnologias Utilizadas

  • Frontend: React e Tailwind CSS
  • Backend: Node.js, Express
  • API: Spotify Web API
  • Autenticação: OAuth 2.0
  • Banco de Dados: PostgreSQL
